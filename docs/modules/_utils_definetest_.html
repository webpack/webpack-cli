    Utility function to run a jscodeshift script within a unit test. This makes several assumptions about the environment.

    Notes:

    • The test should be located in a subdirectory next to the transform itself. Commonly tests are located in a directory called tests.

    • Test data should be located in a directory called testfixtures alongside the transform and tests directory.
